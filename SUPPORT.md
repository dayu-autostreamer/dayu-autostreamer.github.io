# Support

Use the channel that matches the kind of help you need.

## Documentation Website

For broken links, unclear docs, translation issues, missing screenshots, or website build problems, open an issue in this
repository:

<https://github.com/dayu-autostreamer/dayu-autostreamer.github.io/issues>

## Dayu System Usage and Bugs

For installation issues, runtime behavior, scheduler policies, backend/frontend bugs, deployment templates, or feature
requests for the Dayu system itself, open an issue in the system repository:

<https://github.com/dayu-autostreamer/dayu/issues>

## Security

Please do not report vulnerabilities through public issues. Follow [SECURITY.md](SECURITY.md) for private reporting
instructions.
