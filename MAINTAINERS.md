# Dayu Documentation Website Maintainers

This repository is maintained as the public documentation and homepage companion to
[dayu-autostreamer/dayu](https://github.com/dayu-autostreamer/dayu).

## Responsibilities

Website maintainers are responsible for:

- keeping public documentation aligned with Dayu releases and implementation behavior
- reviewing documentation, blog, localization, and Docusaurus configuration pull requests
- keeping English and Simplified Chinese content reasonably synchronized
- triaging website issues and routing system implementation issues to the Dayu repository
- maintaining GitHub Pages deployment and pull request build checks
