Welcome to **Dayu**, an auto scheduling system for stream data processing on distributed cloud-edge platforms.

We provide infrastructure for cloud-edge collaborative stream data analytics and focus on scheduling tasks like configuration optimization, model evolution, task offloading, and so on.
