Stream Data Analysis

Dayu focuses on ai-driven stream data analysis, natively supporting parallel multi-stream processing/monitoring/scheduling with heterogeneous user-defined applications.

Distributed Cloud-Edge Collaboration

Based on a container orchestration framework KubeEdge, dayu is applicable to distributed cloud-edge collaborative scenarios and can be scale to heterogeneous devices.

Adaptive System Scheduling

Dayu can be extended to scheduling topics of different granularity with flexible structure. Adaptive scheduling algorithms thereby can be implemented on dayu for system support.
